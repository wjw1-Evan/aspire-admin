# 自动构建命令

## 描述
编译整个解决方案的后端代码，并构建 Admin 前端项目。

## 模板

### 1. 后端构建
在项目根目录执行 dotnet build，允许警告但阻止错误：
```
dotnet build --no-restore
```

### 2. 前端构建
进入 Platform.Admin 目录，先检查类型错误，再执行构建：
```
cd Platform.Admin
npm run lint
npm run build
```

## 规则
- 先执行后端构建，确保后端代码无编译错误
- 后端构建成功后再执行前端构建
- 前端构建前先运行 lint 进行类型检查
- 如有构建错误，报告错误信息并终止后续操作
- 警告不影响构建流程继续执行
- 构建完成后报告构建结果
