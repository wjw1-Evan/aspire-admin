## 1. 核心实现

### 1.1 统一使用 ToPagedList

所有后端分页查询**必须**使用 `ToPagedList<T>()` 扩展方法：

```csharp
using Platform.ServiceDefaults.Extensions;

public async Task<PagedResult<UserDto>> GetUsersAsync(PageParams request)
{
    // ✅ 正确：使用 ToPagedList 自动处理搜索、排序、分页
    var pagedResult = _context.Set<User>().ToPagedList(request);
    var items = await pagedResult.Queryable.ToListAsync();

    return new PagedResult<UserDto>
    {
        Queryable = items.Select(MapToDto).AsQueryable(),
        CurrentPage = pagedResult.CurrentPage,
        PageSize = pagedResult.PageSize,
        RowCount = pagedResult.RowCount,
        PageCount = pagedResult.PageCount
    };
}
```

### 1.2 ToPagedList 工作流程

```csharp
public static PagedResult<T> ToPagedList<T>(this IQueryable<T> query, PageParams? pageParams)
{
    // 1. 全文搜索（所有 string 字段）
    if (!string.IsNullOrWhiteSpace(pageParams?.Search))
    {
        query = query.Where(BuildSearchFilter<T>(pageParams.Search));
    }

    // 2. 自动排序（默认 createdAt desc）
    query = query.ApplySort(pageParams);

    // 3. 分页
    return query.PageResult(pageParams?.Page ?? 1, pageParams?.PageSize ?? 10);
}
```

## 3. 最佳实践

### 3.1 服务层标准模式

```csharp
public class UserService : IUserService
{
    private readonly ApplicationDbContext _context;

    public async Task<PagedResult<UserDto>> GetUsersAsync(PageParams request)
    {
        var pagedResult = _context.Set<User>()
            .Where(u => u.Status == UserStatus.Active)  // 业务过滤
            .ToPagedList(request);  // 统一分页处理

        var items = await pagedResult.Queryable.ToListAsync();
        var dtos = items.Select(MapToDto).ToList();

        return new PagedResult<UserDto>
        {
            Queryable = dtos.AsQueryable(),
            CurrentPage = pagedResult.CurrentPage,
            PageSize = pagedResult.PageSize,
            RowCount = pagedResult.RowCount,
            PageCount = pagedResult.PageCount
        };
    }
}
```

### 3.2 控制器层

```csharp
[ApiController]
[Route("api/users")]
public class UserController : ControllerBase
{
    private readonly IUserService _userService;

    [HttpPost("list")]
    public async Task<IActionResult> GetUsersList([FromBody] PageParams request)
    {
        var result = await _userService.GetUsersWithRolesAsync(request);
        return Ok(ApiResponse.Success(result));
    }
}
```

### 3.3 复杂查询的处理

对于需要复杂业务逻辑的查询，在 ToPagedList 之前应用业务过滤：

```csharp
public async Task<PagedResult<OrderDto>> GetOrdersAsync(PageParams request, string userId)
{
    var query = _context.Set<Order>()
        .Where(o => o.UserId == userId)
        .Where(o => o.Status != OrderStatus.Cancelled);  // 业务过滤

    // ToPagedList 会自动添加搜索和排序
    var pagedResult = query.ToPagedList(request);

    var items = await pagedResult.Queryable
        .Include(o => o.Items)  // 关联数据
        .ToListAsync();

    return new PagedResult<OrderDto>
    {
        Queryable = items.Select(MapToDto).AsQueryable(),
        CurrentPage = pagedResult.CurrentPage,
        PageSize = pagedResult.PageSize,
        RowCount = pagedResult.RowCount,
        PageCount = pagedResult.PageCount
    };
}
```

## 4. 性能优化

### 4.1 避免 N+1 查询

```csharp
// ✅ 正确：在 ToPagedList 之后 Include
var pagedResult = _context.Set<User>()
    .ToPagedList(request);

var items = await pagedResult.Queryable
    .Include(u => u.Roles)
    .Include(u => u.Department)
    .ToListAsync();
```

### 4.2 选择性字段加载

```csharp
// ✅ 正确：只选择需要的字段
var pagedResult = _context.Set<User>()
    .Select(u => new { u.Id, u.Name, u.Email, u.CreatedAt })
    .ToPagedList(request);

var items = await pagedResult.Queryable.ToListAsync();
```

## 5. 常见问题

### 5.1 搜索不生效

```csharp
// ❌ 错误：搜索字段不是 string 类型
public class Product
{
    public int Id { get; set; }
    public string Name { get; set; }
    public decimal Price { get; set; }  // Price 不会被搜索
}

// ✅ 正确：添加字符串搜索字段
public class Product
{
    public int Id { get; set; }
    public string Name { get; set; }
    public string PriceText => Price.ToString();  // 可搜索
}
```

### 5.2 排序字段不存在

```csharp
// ❌ 错误：排序字段不存在
var result = _context.Set<User>()
    .ToPagedList(new PageParams { SortBy = "invalidField" });

// ✅ 正确：验证排序字段或使用默认排序
public static IQueryable<T> ApplySort<T>(this IQueryable<T> query, PageParams? pageParams)
{
    var sortBy = pageParams?.SortBy;
    if (string.IsNullOrEmpty(sortBy) || !IsValidSortField<T>(sortBy))
    {
        sortBy = "createdAt";  // 默认排序字段
    }
    // ...
}
```
