# 后端分页实现指南

> 统一规范请参阅：[`00-分页规范.md`](../00-分页规范.md)

---

> [!IMPORTANT]
> **所有分页查询必须使用 `ToPagedList()`，禁止手动 `.PageResult()` 或手动 `Skip/Take`。**
> **返回值必须直接返回 `System.Linq.Dynamic.Core.PagedResult<TSource>` 类型，禁止手动创建新的 PagedResult 对象。**

```csharp
using Platform.ServiceDefaults.Extensions;
using System.Linq.Dynamic.Core;

// ✅ 标准写法：直接返回 ToPagedList 结果
var pagedResult = _context.Set<User>()
    .Where(u => u.IsActive)      // 业务过滤
    .ToPagedList(request);       // 自动搜索 + 排序 + 分页

var items = await pagedResult.Queryable.ToListAsync();
pagedResult.Queryable = items.AsQueryable();

return pagedResult;
```

`ToPagedList()` 自动完成以下逻辑：
1. **全文搜索** — 搜索所有 `string` 字段（不区分大小写）。
2. **动态排序** — 按 `SortBy`/`SortOrder` 参数排序。
3. **默认排序** — 如未指定排序参数，默认按 `createdAt desc` 排序。
4. **物理分页** — 按 `Page`/`PageSize` 执行数据库分页。

## 2. 标准模式

### 服务层

```csharp
public async Task<PagedResult<UserDto>> GetUsersAsync(PageParams request)
{
    var pagedResult = _context.Set<User>()
        .Where(u => u.Status == UserStatus.Active)
        .ToPagedList(request);

    var items = await pagedResult.Queryable.ToListAsync();
    return new PagedResult<UserDto>
    {
        Queryable = items.Select(MapToDto).AsQueryable(),
        CurrentPage = pagedResult.CurrentPage,
        PageSize = pagedResult.PageSize,
        RowCount = pagedResult.RowCount,
        PageCount = pagedResult.PageCount
    };
}
```

### 控制器层

```csharp
[HttpGet("list")]
public async Task<IActionResult> GetUsers([FromQuery] PageParams request)
{
    var result = await _userService.GetUsersAsync(request);
    return Success(result);
}
```

> **[强制]** 控制器接收分页参数必须使用 `PageParams`，禁止手动定义 `page`、`pageSize`、`sortBy`、`sortOrder` 等独立参数。

```csharp
// ✅ 正确：使用 PageParams
[HttpGet("list")]
public async Task<IActionResult> GetUsers([FromQuery] PageParams request)

// ❌ 错误：手动定义分页参数
[HttpGet("list")]
public async Task<IActionResult> GetUsers(
    [FromQuery] int page = 1,
    [FromQuery] int pageSize = 10,
    [FromQuery] string? sortBy = null,
    [FromQuery] string? sortOrder = null)
```

## 3. 性能优化建议

> [!TIP]
> - **Include 延迟加载**：将 `Include` 放在 `ToPagedList` 之后，确保仅为当前页的 10-20 条记录加载关联数据，避免笛卡尔积在大数据集上崩溃。
> - **字段投影**：将 `Select` 放在 `ToPagedList` 之前，减少数据库传输的列数。

```csharp
// ✅ 优化 1：Include 放在 ToPagedList 之后
var items = await pagedResult.Queryable
    .Include(u => u.Roles)
    .ToListAsync();

// ✅ 优化 2：Select 放在 ToPagedList 之前
var pagedResult = _context.Set<User>()
    .Select(u => new { u.Id, u.Name, u.Email })
    .ToPagedList(request);
```

## 4. 新建 API 开发规范（强制）

新建分页查询 API 时，必须遵循以下规范：

### 4.1 控制器层

```csharp
// ✅ 正确：使用 [FromQuery] PageParams
[HttpGet("list")]
public async Task<IActionResult> GetItems([FromQuery] PageParams request)
{
    var result = await _service.GetItemsAsync(request);
    return Success(result);
}

// ❌ 错误：手动定义分页参数
[HttpGet("list")]
public async Task<IActionResult> GetItems(
    [FromQuery] int page = 1,
    [FromQuery] int pageSize = 10)
```

### 4.2 服务层

```csharp
// ✅ 正确：直接返回 ToPagedList 结果
public async Task<PagedResult<T>> GetItemsAsync(PageParams pageParams)
{
    var query = _context.Set<Entity>().Where(x => x.IsActive);
    return query.ToPagedList(pageParams);
}

// ❌ 错误：手动实现分页
public async Task<PagedResult<T>> GetItemsAsync(PageParams pageParams)
{
    var items = await query
        .Skip((pageParams.Page - 1) * pageParams.PageSize)
        .Take(pageParams.PageSize)
        .ToListAsync();
    return new PagedResult<T> { ... };
}
```

### 4.3 接口定义（如有需要添加额外参数）

```csharp
// 额外筛选参数放在方法签名中，而非 PageParams
Task<PagedResult<Dto>> GetItemsAsync(PageParams pageParams, bool? isEnabled = null);

// 控制器
public async Task<IActionResult> GetItems(
    [FromQuery] PageParams request,
    [FromQuery] bool? isEnabled = null)
{
    var result = await _service.GetItemsAsync(request, isEnabled);
    return Success(result);
}
```

### 4.4 完整的正确示例

**控制器**：
```csharp
[HttpGet("list")]
[RequireMenu("xxx")]
public async Task<IActionResult> GetItems(
    [FromQuery] PageParams request,
    [FromQuery] string? status = null)
{
    var result = await _itemService.GetItemsAsync(request, status);
    return Success(result);
}
```

**服务层**：
```csharp
public async Task<PagedResult<ItemDto>> GetItemsAsync(PageParams pageParams, string? status = null)
{
    var query = _context.Set<Item>().AsQueryable();

    if (!string.IsNullOrEmpty(status))
    {
        query = query.Where(x => x.Status == status);
    }

    var pagedResult = query.ToPagedList(pageParams);
    var items = await pagedResult.Queryable.ToListAsync();

    return new PagedResult<ItemDto>
    {
        Queryable = items.Select(ToDto).AsQueryable(),
        CurrentPage = pagedResult.CurrentPage,
        PageSize = pagedResult.PageSize,
        RowCount = pagedResult.RowCount,
        PageCount = pagedResult.PageCount
    };
}
```
