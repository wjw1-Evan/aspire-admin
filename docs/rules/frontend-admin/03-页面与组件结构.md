# 页面与组件结构

本规范重点补充前端页面与后端 API 对接时的数据流、接口调用、分页、错误处理等最佳实践。

## 1. 列表页面结构与数据流

```tsx
import { PageContainer, StatCard } from '@/components';
import SearchBar from '@/components/SearchBar';
import { Card, Table } from 'antd';
import { fetchUsers } from '@/services/user/api';

const MyListPage: React.FC = () => {
  const [data, setData] = useState<User[]>([]);
  const [loading, setLoading] = useState(false);
  const [pagination, setPagination] = useState({ page: 1, pageSize: 20, total: 0 });
  const searchParamsRef = useRef<PageParams>({ page: 1, pageSize: 20, search: '' });

  const fetchData = async () => {
    setLoading(true);
    try {
      const res = await fetchUsers(searchParamsRef.current);
      if (res.success && res.data) {
        setData(res.data.queryable || []);
        setPagination(prev => ({ ...prev, total: res.data.rowCount ?? 0 }));
      }
    } finally {
      setLoading(false);
    }
  };

  return (
    <PageContainer style={{ paddingBlock: 12 }}>
      {/* 统计卡片区域 */}
      <Card style={{ marginBottom: 16 }}>
        <Row gutter={[12, 12]}>
          <Col xs={24} sm={12} md={6}>
            <StatCard title="总数" value={pagination.total} icon={<Icon />} />
          </Col>
        </Row>
      </Card>

      {/* 搜索表单 */}
      <SearchBar
        style={{ marginBottom: 16 }}
        onSearch={(params) => {
          searchParamsRef.current = { ...searchParamsRef.current, ...params, page: 1 };
          fetchData();
        }}
      />

      {/* 数据表格 */}
      <Table
        dataSource={data}
        columns={columns}
        rowKey="id"
        loading={loading}
        pagination={{
          current: pagination.page,
          pageSize: pagination.pageSize,
          total: pagination.total,
          onChange: (page, pageSize) => {
            searchParamsRef.current = { ...searchParamsRef.current, page, pageSize };
            fetchData();
          },
        }}
      />
    </PageContainer>
  );
};
```

## 3. 统一分页与后端对接

- 所有分页表格组件必须严格对齐后端 PagedResult 结构，参数命名统一为 page/pageSize。
- 分页、排序、筛选参数需与后端接口保持一致。

## 4. 错误处理与用户提示

- 所有接口调用必须判断 res.success，失败时统一 message.error 提示。
- 401/403/500 等异常由 request 拦截器统一处理。

## 5. 变更与维护建议

- 页面与组件如需扩展，优先遵循后端 API 结构和统一数据流。
- 发现实现与本规范或后端标准不符时，优先以本规范为准，及时修订文档与代码。

## 组件拆分

### 拆分原则

- 复用较高的表单、列表操作栏等拆分为独立组件
- 放在 `src/components/` 或 `pages/xxx/components/` 目录
- UI 组件不直接调用后端 services

```tsx
// src/pages/task-management/hooks/useTaskManagement.ts
import { useState, useCallback } from 'react';
import { queryTasks, deleteTask } from '@/services/task/api';
import type { TaskDto, TaskStatistics } from '../types';

export function useTaskManagement() {
  const [loading, setLoading] = useState(false);

  const fetchTasks = useCallback(async (params) => {
    setLoading(true);
    try {
      const response = await queryTasks(params);
      return response;
    } finally {
      setLoading(false);
    }

      await deleteTask(taskId);
      return true;
    } finally {
      setLoading(false);
    }
  }, []);

  return {
    loading,
    statistics,
    fetchTasks,
    handleDelete,
  };
}
```

## 基准页面结构（以 PasswordBook 模块为例）

本项目将 `src/pages/password-book` 模块定义为**前端复杂列表页面的基准实现模板**。基于"高内聚低耦合"与实用主义的原则，避免因过度拆分导致零碎文件过多（Fragmented Horror），我们推崇**智能组装的单文件模式**。

### 目录结构

```
src/pages/password-book/
├── index.tsx              # 主页面（核心业务逻辑，≤400行）
├── types.ts               # 类型定义（放在页面同级目录）
└── components/            # 仅当组件会被其他业务页面复用时才建立
    ├── PasswordBookForm.tsx
    ├── PasswordGenerator.tsx
    └── ExportDialog.tsx
```

### 实战规范守则

1. **拒绝无意义的文件拆分**：
   - 除非一个子组件/钩子明显会在**其他业务页面**被复用，否则**坚决不建立单独的文件**。
   - 不要仅仅为了把主文件缩短而把页面内的子结构（如统计区、详情抽屉）生硬地剔出到 `components` 目录下。

2. **使用原生 Ant Design 组件**：
   - 列表使用原生 `<Table>` 组件，不要使用封装的 CrudTable。
   - 操作列（查看/编辑/删除）直接在 `columns` 中定义，删除使用 `modal.confirm` 确认。
   - 原生组件能减少认知负担，团队新人更容易上手。

3. **提高单文件内部逻辑密度**：
   - 不超过 **400行 - 500行**。
   - 使用循环等代码技巧来生成重复的静态 UI 结构（例如使用 `Array.map` 生成上方的四个统计卡片，或者配置项动态渲染详情的 `Descriptions`）。

4. **类型定义就近原则**：
   - 页面级类型放在页面目录的 `types.ts` 中，而非 src/types/ 深处。
   - 使用 `PagedResult<T>` 包装分页响应类型。

5. **服务层独立文件**：
   - API 调用放在 `src/services/xxx/api.ts` 中。
   - 每个 API 函数独立导出，不合并为对象。

```tsx
// src/pages/password-book/index.tsx
import React, { useRef, useState, useCallback, useEffect } from 'react';
import { PageContainer, StatCard } from '@/components';
import SearchBar from '@/components/SearchBar';
import { useIntl } from '@umijs/max';
import { Grid, Tag, Space, Modal, Drawer, Row, Col, Card, Input, Descriptions, Spin, App, Button, Table } from 'antd';
import type { ColumnsType } from 'antd/es/table';
import {
  PlusOutlined, ExportOutlined, ReloadOutlined, LockOutlined, FolderOutlined,
  ClockCircleOutlined, GlobalOutlined, UserOutlined, KeyOutlined, TagOutlined,
  CalendarOutlined, CopyOutlined, EditOutlined, DeleteOutlined,
} from '@ant-design/icons';
import dayjs from 'dayjs';
import { getPasswordBookList, deletePasswordBookEntry, getPasswordBookStatistics, getPasswordBookEntry } from '@/services/password-book/api';
import type { PageParams } from '@/types/page-params';
import type { PasswordBookEntry, PasswordBookEntryDetail, PasswordBookStatistics } from './types';
import PasswordBookForm from './components/PasswordBookForm';
import ExportDialog from './components/ExportDialog';
import useCommonStyles from '@/hooks/useCommonStyles';

const PasswordBook: React.FC = () => {
  const intl = useIntl();
  const { message, modal } = App.useApp();
  const { styles } = useCommonStyles();
  const isMobile = !Grid.useBreakpoint().md;

  // 状态定义
  const [data, setData] = useState<PasswordBookEntry[]>([]);
  const [loading, setLoading] = useState(false);
  const [pagination, setPagination] = useState({ page: 1, pageSize: 10, total: 0 });
  const [statistics, setStatistics] = useState<PasswordBookStatistics | null>(null);
  const [formVisible, setFormVisible] = useState(false);
  const [detailVisible, setDetailVisible] = useState(false);
  const [exportVisible, setExportVisible] = useState(false);
  const [editingEntry, setEditingEntry] = useState<PasswordBookEntryDetail | null>(null);
  const [viewingEntry, setViewingEntry] = useState<PasswordBookEntryDetail | null>(null);

  const searchParamsRef = useRef<PageParams>({ page: 1, pageSize: 10, search: '' });

  // 核心拉取逻辑
  const fetchData = useCallback(async () => {
    setLoading(true);
    try {
      const response = await getPasswordBookList(searchParamsRef.current);
      if (response.success && response.data) {
        setData(response.data.queryable || []);
        setPagination(prev => ({
          ...prev,
          page: searchParamsRef.current.page ?? prev.page,
          pageSize: searchParamsRef.current.pageSize ?? prev.pageSize,
          total: response.data.rowCount ?? 0,
        }));
      } else {
        setData([]);
        setPagination(prev => ({ ...prev, total: 0 }));
      }
    } finally {
      setLoading(false);
    }
  }, []);

  const fetchStatistics = useCallback(async () => {
    const res = await getPasswordBookStatistics();
    if (res.success && res.data) setStatistics(res.data);
  }, []);

  const refreshAll = useCallback(() => {
    fetchData();
    fetchStatistics();
  }, [fetchData, fetchStatistics]);

  useEffect(() => { refreshAll(); }, [refreshAll]);

  // 行操作事件
  const handleView = async (entry: PasswordBookEntry) => {
    const res = await getPasswordBookEntry(entry.id);
    if (res.success && res.data) {
      setViewingEntry(res.data);
      setDetailVisible(true);
    } else {
      message.error('获取详情失败');
    }
  };

  const handleEdit = async (entry: PasswordBookEntry) => {
    const res = await getPasswordBookEntry(entry.id);
    if (res.success && res.data) {
      setEditingEntry(res.data);
      setFormVisible(true);
    } else {
      message.error('获取详情失败');
    }
  };

  const handleDelete = (entry: PasswordBookEntry) => {
    modal.confirm({
      title: '确认删除',
      content: '不可逆操作，确定要删除此条目吗？',
      okText: '确定',
      cancelText: '取消',
      okButtonProps: { danger: true },
      onOk: async () => {
        const res = await deletePasswordBookEntry(entry.id);
        if (res.success) {
          message.success('删除成功');
          refreshAll();
        }
      },
    });
  };

  // 统计块配置（数组驱动渲染）
  const statsConfig = [
    { title: '总条目数', value: statistics?.totalEntries, icon: <LockOutlined />, color: '#1890ff' },
    { title: '分类数量', value: statistics?.categoryCount, icon: <FolderOutlined />, color: '#52c41a' },
    { title: '标签数量', value: statistics?.tagCount, icon: <TagOutlined />, color: '#722ed1' },
    { title: '最近使用（7天）', value: statistics?.recentUsedCount, icon: <ClockCircleOutlined />, color: '#faad14' },
  ];

  // 列表列配置
  const columns: ColumnsType<PasswordBookEntry> = [
    { title: '平台', dataIndex: 'platform', key: 'platform', sorter: true, render: (text, record) => <a onClick={() => handleView(record)}>{text}</a> },
    { title: '账号', dataIndex: 'account', key: 'account', sorter: true },
    { title: '网址', dataIndex: 'url', key: 'url', render: (url) => url ? <a href={url} target="_blank">{url}</a> : '-' },
    { title: '分类', dataIndex: 'category', key: 'category', render: (c) => c ? <Tag color="blue">{c}</Tag> : '-' },
    { title: '标签', dataIndex: 'tags', key: 'tags', render: (tags) => tags?.length ? <Space wrap>{tags.map(t => <Tag key={t}>{t}</Tag>)}</Space> : '-' },
    { title: '最后使用', dataIndex: 'lastUsedAt', key: 'lastUsedAt', sorter: true, render: (d) => d ? new Date(d).toLocaleString() : '-' },
    {
      title: '操作', key: 'actions', fixed: 'right', width: 120,
      render: (_, record) => (
        <Space size="small">
          <Button type="link" size="small" icon={<EditOutlined />} onClick={() => handleEdit(record)}>编辑</Button>
          <Button type="link" size="small" danger icon={<DeleteOutlined />} onClick={() => handleDelete(record)}>删除</Button>
        </Space>
      ),
    },
  ];

  return (
    <PageContainer
      title={<Space><LockOutlined />密码本</Space>}
      style={{ paddingBlock: 12 }}
      extra={
        <Space wrap>
          <Button icon={<ReloadOutlined />} onClick={refreshAll}>刷新</Button>
          <Button icon={<ExportOutlined />} onClick={() => setExportVisible(true)}>导出</Button>
          <Button type="primary" icon={<PlusOutlined />} onClick={() => { setEditingEntry(null); setFormVisible(true); }}>新建</Button>
        </Space>
      }
    >
      {/* 1. 统计区域：数组循环渲染 */}
      {statistics && (
        <Card className={styles.card} style={{ marginBottom: 16 }}>
          <Row gutter={[12, 12]}>
            {statsConfig.map((stat, idx) => (
              <Col xs={24} sm={12} md={6} key={idx}>
                <StatCard title={stat.title} value={stat.value ?? 0} icon={stat.icon} color={stat.color} />
              </Col>
            ))}
          </Row>
        </Card>
      )}

      {/* 2. 搜索表单 */}
      <SearchBar
        initialParams={searchParamsRef.current}
        style={{ marginBottom: 16 }}
        onSearch={(params) => { searchParamsRef.current = { ...searchParamsRef.current, ...params, page: 1 }; fetchData(); }}
      />

      {/* 3. 数据表格 */}
      <Table<PasswordBookEntry>
        dataSource={data}
        columns={columns}
        rowKey="id"
        loading={loading}
        scroll={{ x: 'max-content' }}
        onChange={(pag, _, sorter) => {
          searchParamsRef.current = {
            ...searchParamsRef.current,
            page: pag.current,
            pageSize: pag.pageSize,
            sortBy: sorter?.field,
            sortOrder: sorter?.order === 'ascend' ? 'asc' : sorter?.order === 'descend' ? 'desc' : undefined,
          };
          fetchData();
        }}
        pagination={{
          current: pagination.page,
          pageSize: pagination.pageSize,
          total: pagination.total
         
        }}
      />

      {/* 4. 创建/编辑弹窗 */}
      <Modal
        title={editingEntry ? '编辑' : '新建'}
        open={formVisible}
        onCancel={() => { setFormVisible(false); setEditingEntry(null); }}
        footer={null}
        width={isMobile ? '100%' : 600}
        destroyOnHidden
      >
        <PasswordBookForm
          entry={editingEntry}
          onCancel={() => { setFormVisible(false); setEditingEntry(null); }}
          onSuccess={() => { setFormVisible(false); setEditingEntry(null); refreshAll(); }}
        />
      </Modal>

      {/* 5. 详情抽屉 */}
      <Drawer
        title="详情"
        placement="right"
        open={detailVisible}
        onClose={() => { setDetailVisible(false); setViewingEntry(null); }}
        size={isMobile ? '100%' : 600}
      >
        <Spin spinning={loading}>
          {viewingEntry && (
            <>
              <Card title="基本信息" style={{ marginBottom: 16 }}>
                <Descriptions column={isMobile ? 1 : 2} size="small">
                  <Descriptions.Item label="平台"><strong>{viewingEntry.platform}</strong></Descriptions.Item>
                  <Descriptions.Item label="账号">{viewingEntry.account}</Descriptions.Item>

                  <Descriptions.Item label="更新时间">{viewingEntry.updatedAt}</Descriptions.Item>

      </Drawer>

      <ExportDialog open={exportVisible} onClose={() => setExportVisible(false)} />
    </PageContainer>
  );
};
```

```ts
// src/services/password-book/api.ts
import { request } from '@umijs/max';
import type { ApiResponse, PagedResult } from '@/types/unified-api';
import type { PageParams } from '@/types/page-params';
import type { PasswordBookEntry, PasswordBookEntryDetail, PasswordBookStatistics, CreatePasswordBookEntryRequest, UpdatePasswordBookEntryRequest } from '@/pages/password-book/types';

export async function createPasswordBookEntry(data: CreatePasswordBookEntryRequest) {
  return request<ApiResponse<PasswordBookEntry>>('/api/password-book', { method: 'POST', data });
}

export async function updatePasswordBookEntry(id: string, data: UpdatePasswordBookEntryRequest) {
  return request<ApiResponse<PasswordBookEntry>>(`/api/password-book/${id}`, { method: 'PUT', data });
}

export async function getPasswordBookEntry(id: string) {
  return request<ApiResponse<PasswordBookEntryDetail>>(`/api/password-book/${id}`, { method: 'GET' });
}

export async function getPasswordBookList(params: PageParams) {
  return request<ApiResponse<PagedResult<PasswordBookEntry>>>('/api/password-book/list', { method: 'GET', params });
}

export async function deletePasswordBookEntry(id: string) {
  return request<ApiResponse<void>>(`/api/password-book/${id}`, { method: 'DELETE' });
}

export async function getPasswordBookStatistics() {
  return request<ApiResponse<PasswordBookStatistics>>('/api/password-book/statistics', { method: 'GET' });
}
```
