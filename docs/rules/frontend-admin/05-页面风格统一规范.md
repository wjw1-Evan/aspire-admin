# 页面风格统一规范

> **[强制]** 所有列表/管理页面必须遵循以下统一风格。

## 页面容器

### PageContainer 设置
```tsx
<PageContainer style={{ paddingBlock: 12 }}>
  {/* 页面内容 */}
</PageContainer>
```

- 统一使用 `paddingBlock: 12`
- **禁止**使用自定义 `padding` 或 `margin` 覆盖默认样式

## 统计卡片区域

### 必须使用 StatCard
```tsx
import { StatCard } from '@/components';
import { Card, Row, Col } from 'antd';

<Card style={{ marginBottom: 16, borderRadius: 12 }}>
  <Row gutter={[12, 12]}>
    <Col xs={24} sm={12} md={6}>
      <StatCard
        title="总任务数"
        value={statistics.totalTasks}
        icon={<TaskIcon />}
        color="#1890ff"
      />
    </Col>
    <Col xs={24} sm={12} md={6}>
      <StatCard
        title="进行中"
        value={statistics.inProgressTasks}
        icon={<ClockCircleIcon />}
        color="#52c41a"
      />
    </Col>
  </Row>
</Card>
```

### StatCard 组件特性
- 图标在左，数值和标题在右垂直排列
- 卡片内边距：`10px 12px`
- 图标尺寸：`20px`
- 数值字号：`20px`
- 标题字号：`12px`

### 响应式断点
- 移动端 `xs={24}`：单列
- 平板 `sm={12}`：两列
- 桌面 `md={6}`：四列

## 数据表格

### 统一使用 DataTable
- 所有列表页面必须使用 `DataTable` 组件
- 操作列固定 150px，固定在右侧

### 操作列规范
```tsx
{

  valueType: 'option',
  width: 150,
  fixed: 'right',
  render: (_, record) => (
    <Space size="small">
```

## 样式规范速查

### 间距规范
| 元素 | 间距 |
|------|------|
| 区块内边距 | 16px |
| 分区间距 | 12px |
| 表单项上下 | 12px |
| 按钮组间距 | 8px |
| 列表顶部/底部留白 | 12px |

### 文字规范
| 元素 | 字号/字重 |
|------|------------|
| 一级标题 | 24px / 600 |
| 卡片标题 | 16px / 600 |
| 正文 | 14px / 400 |
| 次要文字 | 12px / 400 |

### 颜色规范
- 主色：使用 AntD token `colorPrimary`
- **禁止**使用自定义全局色板
- 状态色：统一使用 AntD 语义色 `success` / `info` / `warning` / `error`

### 组件规范
| 组件 | 规范 |
|------|------|
| 卡片 | `<Card bordered={false}>` |
| 表单按钮 | 固定在底部吸底区或抽屉底部 |
| 删除按钮 | 必须使用 `Popconfirm` 确认 |
| 图标 | 使用 `@ant-design/icons`，尺寸 14/16px |

## 常见错误示例

### ❌ 错误 1：自定义全局样式
```tsx
// ❌ 禁止：自定义全局 CSS class
<div className="my-custom-class">

// ✅ 正确：使用 AntD 组件和内联样式
<div style={{ padding: 16 }}>
```

### ❌ 错误 2：表格未固定操作列
```tsx
// ❌ 禁止
{
  title: '操作',
  // 缺少 fixed: 'right'
}

// ✅ 正确
{
  title: '操作',
  fixed: 'right',
  width: 150,
}
```

### ❌ 错误 3：删除未使用 Popconfirm
```tsx
// ❌ 禁止
<Button danger onClick={handleDelete}>删除</Button>

// ✅ 正确
<Popconfirm title="确认删除？" onConfirm={handleDelete}>
  <Button danger>删除</Button>
</Popconfirm>
```

### ❌ 错误 4：自定义主色
```tsx
// ❌ 禁止：自定义主色
<div style={{ color: '#ff0000' }}>

// ✅ 正确：使用 AntD token
<div style={{ color: 'var(--ant-primary-color)' }}>
```
