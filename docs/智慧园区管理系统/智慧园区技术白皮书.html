CHAPTER 01

核心架构：云原生与整洁架构哲学

系统采用先进的云原生分布式架构，旨在提供高性能、高可用且具备极致扩展性的业务底座。

PLATFORM .NET 10 全栈技术体系

作为业界首批拥抱 .NET 10 的企业级平台，系统深度利用了新一代运行时的 AOT 编译优化与内存管理特性：

  • 极致性能：原生支持 Native AOT，冷启动速度提升 80%，运行期内存占用降低 40%。
  • 跨平台一致性：基于统一的运行时，确保从 Linux 容器到边缘网算的无缝迁移。
  • 现代语法特性：充分利用 C# 14 的增强模式匹配与性能改进，提升代码健壮性。

ORCHESTRATION .NET Aspire 分布式编排

通过 .NET Aspire，我们将复杂的分布式系统开发简化为“应用组合”模式：

  • 全栈遥测支持：内置 OpenTelemetry 协议，实现端到端的分布式追踪 (Tracing) 与指标监控 (Metrics)。
  • 服务自发现：基于组件化的服务连接管理，无需手动配置复杂的连接字符串。
  • 本地生产一致性：开发环境与线上容器环境使用统一的项目配置逻辑，极大降低“环境偏差”导致的 Bug。
// Aspire AppHost 编排示例

PATTERN 整洁架构 (Clean Architecture)

我们严格遵循整洁架构原则，确保业务逻辑 (Domain) 与技术细节 (Infrastructure) 彻底解耦：

  • 领域驱动设计 (DDD)：以园区业务实体为核心，确保系统演进不被数据库或框架绑架。
  • 依赖注入与抽象：所有的基础设施（如存储、通知、AI 接口）均通过接口定义，支持分钟级的插件化替换。
  • 高可测试性：业务逻辑层不依赖任何外部组件，单元测试覆盖率可轻松达到 95% 以上。
CHAPTER 02

物联网 (IoT) 平台：海量接入与实时决策

园区万物互联的基石，提供从设备感知、数据传输到逻辑联动的一站式技术支撑。

MESSAGING 高性能 MQTT 消息路由

基于高度优化的 MQTT 协议栈，系统支持海量并发连接与亚秒级消息路由：

  • 百万级长连接：采用流式处理模型，单节点可支撑 10W+ 设备同时在线，支持集群横向扩展。
  • QoS 等级保障：针对告警、状态、配置等不同数据流，提供精细化的消息质量保障机制。
  • 异构设备穿透：内置 50+ 种标准工业与楼宇协议驱动（Modbus, BACnet, Zigbee 等），实现真正的“即插即用”。

LOGIC 动态规则引擎与流计算

系统内置轻量级流式计算引擎，支持复杂的设备联动逻辑与实时告警：

  • 可视化编排：非技术人员可通过拖拽式界面定义“如果...则...”的自动化场景。
  • 边缘侧计算支持：核心逻辑可下发至边缘网关，确保在断网状态下园区关键安防逻辑依然有效。
  • 数据清洗与聚合：在数据入库前进行实时的异常过滤（Noise Filtering）与聚合统计，大幅降低海量数据存储压力。

TWIN 全要素数字孪生 (Digital Twin)

不仅是 3D 展示，更是具备逻辑深度的数据镜像：

  • 状态实时同步：物理空间与虚拟模型间的时延控制在 200ms 以内。
  • 全生命周期管理：记录设备从巡检、维修到报废的每一次状态变更。
  • 空天内容映射：支持从空间坐标到业务逻辑的深度映射，实现“点选设备，即出报表”。
CHAPTER 03

AI Agent：基于 MCP 协议的主动自治

引入 Model Context Protocol (MCP)，将 AI 从简单的问答助手提升为具备环境感知力的自动代理。

PROTOCOL MCP 开放协议深度集成

系统作为 MCP Host，允许大模型直接理解并调用园区的各类动态数据源：

  • 上下文无缝注入：无需频繁通过 API 调用，AI 逻辑可实时通过 MCP 获取最新的设备拓扑与运行指标。
  • 跨模型兼容性：支持 Claude、GPT 等主流模型作为推理引擎，灵活适配不同业务场景。
  • 标准化资源访问：将园区资源（温控、照明、工单）抽象为标准的 MCP 资源，实现技术层面的语义一致性。

COGNITION 从被动响应到主动代理

AI 不再只是等待指令，而是通过长短期记忆模块 (Memory) 实现自主决策：

  • 自适应策略优化：基于历史用能数据，AI 自动生成节能方案并征得确认后执行（Human-in-the-loop）。
  • 故障预研判：结合 IoT 传感器波动，AI 自动发起预警并关联生成维护工单。
  • 自然语言交互界面：运营人员可使用自然语言进行复杂的策略配置：“当南区人流超过 200 人时，调高空调新风量”。
CHAPTER 04

企业级安全与合规：全栈可信体系

安全不是补丁，而是贯穿系统生命周期的原生属性。我们提供行业领先的合规性与数据安全保障。

ENCRYPTION 国密商用密码算法集成

系统深度适配国家密码管理局标准，确保关键业务数据的机密性与完整性：

  • SM2 (非对称加密)：用于身份鉴别、数字签名与密钥交换，性能优于传统 RSA。
  • SM3 (杂凑算法)：确保文件与日志的一致性校验，防篡改、抗碰撞。
  • SM4 (对称加密)：用于海量 IoT 数据与静态数据库的实时加解密，提供硬件级的吞吐性能。

COMPLIANCE 网络安全等级保护 2.0 (三级等保)

系统架构设计完全符合三级等保的高标准要求：

  • 多级访问控制：基于 RBAC + ABAC 的精细化权限模型，实现真正的权责分离。
  • 全过程审计记录：从运维操作到 AI 决策，每一条指令均具备不可抵赖的可追溯性。
  • 物理与逻辑隔离：支持多租户独立数据库架构，物理层实现关键业务数据的硬隔离。
